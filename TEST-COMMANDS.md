Commands used in Jekyll:

| Syntax | Use |
|-------|--------|---------|
| jekyll new PROJECT_SITE_NAME | Initialize the Jekyll site. |
| bundle exec jekyll serve | Build the site and make it available on a local server, with gemfile packages. |
| jekyll serve | Build the site and make it available on a local server. |
| jekyll serve --draft | Build the site and make it available on a local server, including drafts. |
| bundle install | Installs gems in the Gemfile. |
| bundle update | Update gems in the Gemfile. Used when adding new gems.  |
| SYNTAX | USE |
